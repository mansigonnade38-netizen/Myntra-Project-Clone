# Myntra-Project-Clone


## 📌 Project Overview

This project is a **Myntra website clone** developed using **HTML, CSS, and JavaScript**.
It replicates the basic layout and functionality of the Myntra e-commerce platform such as product display, navigation bar, and shopping features.

The purpose of this project is to practice **front-end web development skills** and understand how real e-commerce websites are structured.

---

## 🚀 Features

* Responsive navigation bar
* Product listing section
* Add to cart functionality
* Stylish UI similar to Myntra
* Interactive buttons using JavaScript
* Clean and responsive design

---

## 🛠️ Technologies Used

* **HTML5** – For creating the website structure
* **CSS3** – For styling and layout design
* **JavaScript** – For dynamic functionality and user interaction

---

---

## 💻 How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in any web browser.

The Myntra clone website will run locally on your system.
